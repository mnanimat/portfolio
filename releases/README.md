# Distribuições

- `MNPortfolio-Unreal-Windows-v1.0.zip`: aplicativo Unreal Engine para Windows. Extraia o ZIP e execute `Install-MNPortfolio.cmd`.
- O editor de modelagem e animação Motion Forge pode ser instalado diretamente pela versão web pública usando **Instalar aplicativo** no Edge ou Chrome.

Produção: <https://mn-animation-3d-portfolio.mnanimat.chatgpt.site>
